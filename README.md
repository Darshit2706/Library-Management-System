# 📚 Library Management System (C++)

A simple **Console-Based Library Management System** built using C++.  
This project allows users to manage books with features like adding, issuing, returning, searching, and deleting books.  
All data is stored in a local file (`books.txt`) for persistence.

---

## 🚀 Features

- Add new books
- Display all books
- Search book by ID
- Issue a book
- Return a book
- Delete a book
- File handling (data saved in `books.txt`)
- Persistent storage (data remains after program exit)

---

## 🛠️ Technologies Used

- C++
- File Handling (`fstream`)
- STL Vector
- OOP (Classes & Objects)

---

## 📁 Project Structure
main.cpp # Source code
books.txt # Data file (auto-created)


---

## ⚙️ How to Run

### 1. Compile the program
Use any C++ compiler like g++:

```bash
g++ main.cpp -o library
```
